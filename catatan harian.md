Selama 2 minggu ini saya akan berlatih tentang 

* Cara kerja dan perbedaan antara OSI dan TCP/IP 
* Memahami protokol http/s, dns, ssh, ftp, smb
* wireshark
* SIEM (splunk)

Yang akan aku pelajari dari protokol adalah  : 

Serangan apa yang biasa terjadi di protokol http/s, dns, ssh, ftp, smb dan cara menangani serangan  itu.

contoh : 
* HTTP/S : Paham bedanya status code **220 OK** (berhasil), **404 Not Found** (mungkin direktory scanning), dan **403 Forbidden**.
* DNS : Paham apa itu DNS Tunneling (mengirim data lewat protocol dns agar tidak terdeteksi wirewall).
* SSH/RDP : Paham cara membedakan Brute Force (ribuan kali percobaan login gagal dari satu IP dalam waktu singkat) dan Credential Stuffing.
