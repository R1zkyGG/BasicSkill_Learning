## Tcp/IP


---

**TCP/IP** (Transmission Control Protocol/Internet Protocol) adalah "bahasa" atau standar komunikasi yang digunakan oleh komputer di seluruh dunia untuk saling bertukar data melalui internet atau jaringan lokal.


---

Berbeda dengan OSI layer yang memiliki 7 layer, TCP/IP hanya memiliki 4 layer 

lapisan utamanya :

Layer 4 Application 
- Funsi utama : Tempat aplikasi berinteraksi dengan jaringan.
- Contoh protokol : HTTP (Web), SMTP (Email), FTP (File)

Layer 3 Trasport 
- Fungsi utama : Mengatur komunikasi antar host dan memastikan data utuh.
- Contoh protokol : TCP,UDP

Layer 2 Internet 
- Fungsi utama : Menangani pengalamatan dan rute perjalanan paket.
- Contoh protokol : IP (IPv4/IPv6), ICMP (Ping)

Layer 1 Network Access
- Fungsi utama : Mengurus fisik (kabel/Wi-Fi) dan pengiriman data di media.
- Contoh protokol : Ethernet, Wi-Fi (802.11)
